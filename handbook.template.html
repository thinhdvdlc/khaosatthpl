A · Deploy (VNPT SmartCloud)

Dockerfile→image→Container Registry→Helm→VKS→Ingress
docker build -t registry.smartcloud.vn/<team>/__SLUG__:dev .
docker push registry.smartcloud.vn/<team>/__SLUG__:dev
helm upgrade --install __SLUG__ deploy/helm -n __SLUG__ --create-namespace \
  --set image.repository=registry.smartcloud.vn/<team>/__SLUG__ --set image.tag=dev \
  --set existingSecret=__SLUG__-secret --wait
Managed: PostgreSQL · Redis · Object Storage S3 (s3hn.smartcloud.vn). CI/CD: push development → build image + helm upgrade.

B · Quy chuẩn Prompt

Nguyên tắc "Context Engineering"

  • Vai trò: BA / Dev / QA / DevOps.
  • Phạm vi cho phép/cấm: file được đụng, việc không được làm.
  • Nguồn sự thật: trỏ docs/ (problem, PRD, architecture).
  • Definition of Done: test · health · migration · evidence.
